# ScholarBridge Architecture

## Overview
ScholarBridge is a Next.js SaaS platform for graduate academic support. It enables students, experts, administrators, and quality reviewers to collaborate securely around research projects, milestones, documents, and payments.

## Core modules
- Marketing site
- Student dashboard
- Expert workspace
- Admin console
- API health endpoint
- Prisma data layer

## Deployment
- Frontend: Vercel
- Database: Neon PostgreSQL
- ORM: Prisma
- CI/CD: GitHub Actions
